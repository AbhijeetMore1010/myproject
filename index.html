Konakage — Simple automation for developers

Lightweight tool to manage workflows, deployments and small automations. Designed for hosting and DevOps use-cases.

Get Started

Easy Setup

Clone, configure a few env vars and run — minimal dependencies.

Small footprint

Optimized for low-resource servers (works on t2.micro / small VPS).

DevOps friendly

Includes scripts for backups, deploys and health checks.

Install (example)

git clone https://github.com/yourusername/konakage.git
cd konakage
# for PHP/Laravel example
composer install
cp .env.example .env
php artisan key:generate
php artisan serve
